# 上班助手 - 项目进度记录

## 项目概述
开发一款桌面端上班助手软件，帮助用户建立明确的时间观念，提高工作效率。

### 运行方式

```bash
# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# 安装依赖
pip install -r requirements.txt

# 运行应用
python main.py
```
